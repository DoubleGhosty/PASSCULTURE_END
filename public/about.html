À propos du projet

PASS est une application full-stack développée dans le cadre du cours Digital Ecosystem (ILAC × EFREI, Spring 2026). Elle permet aux utilisateurs de se connecter avec leurs identifiants Pass Culture et de consulter leur profil, leur crédit restant, et leur historique de réservations culturelles depuis une interface web moderne.

Le projet démontre une architecture cloud-native complète : intégration d'APIs externes, orchestration côté serveur, stockage persistant dans une base de données managée, et déploiement entièrement automatisé sur des plateformes cloud.

Toutes les clés API sont stockées dans des variables d'environnement côté serveur. Aucun secret n'est exposé dans le code source ou dans le navigateur.
